# إعداد المشروع

تم تنفيذ هذا المشروع بواسطة:
- محمد عمرو السيد
- باسم عمرو السيد
- عبدالرحمن عمرو السيد

## Front-End (REACT)

### تفاصيل التثبيت

1. الانتقال إلى مجلد المشروع:
   ```
   cd Front-End (REACT)
   ```

2. تثبيت التبعيات:
   باستخدام npm:
   ```
   npm install
   ```


3. تشغيل الواجهة الأمامية للتطوير:
   ```
   npm start
   ```

4. بناء نسخة الإنتاج:
   ```
   npm run build
   ```
