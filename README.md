# إعداد المشروع

تم تنفيذ هذا المشروع بواسطة:
- محمد عمرو السيد
- باسم عمرو السيد
- عبدالرحمن عمرو السيد

## Front-End (REACT)

### تفاصيل التثبيت

1. الانتقال إلى مجلد المشروع:
   ```
   cd Front-End (REACT)
   ```

2. تثبيت التبعيات:
   باستخدام npm:
   ```
   npm install
   ```


3. تشغيل الواجهة الأمامية للتطوير:
   ```
   npm start
   ```

4. بناء نسخة الإنتاج:
   ```
   npm run build
   ```

## Back-End (python)

### تفاصيل التثبيت

1. الانتقال إلى مجلد المشروع:
   ```
   cd Back-End (python)
   ```

2. إعداد البيئة الافتراضية (اختياري ولكن موصى به):
   ```
   python -m venv venv
   source venv/bin/activate  # على نظام Unix
   # أو
   venv\Scripts\activate  # على نظام Windows
   ```

3. تثبيت التبعيات:
   ```
   pip install -r requirements.txt
   ```

4. تشغيل خادم Flask:
   ```
   flask run
   ```
   أو
   ```
   python main_server.py
   ```
